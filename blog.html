1) What is the difference between LocanStorage and SessionStorage?

2) What is the difference between Global Scope and Block Scope?

Ans: Value inside a global scope can be accessed from anywhere of a JavaScript file. But values inside a block scope can be accessed only from the particular part or area in a JavaScript file.


3) What is JavaScript event loop?

Ans: Event loop is an endless loop, which waits for tasks, executes them and then sleeps until it receives more tasks. The event loop executes tasks from the event queue only when the call stack is empty.


4) How many ways are available to get undefined in JavaScript?

Ans: There are 8 ways to get undefined in JavaScript. these are:
  1. Variable that is not initialized will through undefined
  2. Function with nothing in return will give undefined
  3. Parameter that is not passed will be undefined
  4. When a property inside an object does not exist will give undefined
  5. Accessing an array element out of its index range will give undefined
  6. Deleting an element inside an array
  7. Declaring a variable to undefined
